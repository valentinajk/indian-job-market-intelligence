Indian Job Market Intelligence Dashboard

Overview
This project is an end-to-end data analytics and machine learning application built to analyze salary trends in the Indian job market. The system processes over 97,000 job postings, performs data cleaning and feature engineering, and provides interactive salary insights through a Streamlit web application.

Key Features
- Market overview metrics
- Tech vs non-tech salary comparison
- Salary growth by experience
- Top paying cities analysis
- Interactive filtering by city and role type
- Salary prediction tool

Machine Learning
- Linear Regression model
- Random Forest model
- Feature importance analysis
- Iterative model improvement and evaluation

Data Processing
- Handling missing values
- Outlier removal using IQR method
- Feature engineering (average salary and experience)
- Role classification (tech vs non-tech)

Technology Stack
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Streamlit

Purpose
The goal of this project is to demonstrate real-world data cleaning, exploratory data analysis, machine learning modeling, and deployment of an interactive analytics dashboard.
